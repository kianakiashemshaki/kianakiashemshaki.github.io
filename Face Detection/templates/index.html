Project Description

Hi, I'm Kiana Kiashemshaki. This project employs Python, Flask, OpenCV, and the MTCNN algorithm to detect faces in uploaded images. Users can upload an image through the web interface, and the application will promptly highlight detected faces by drawing rectangles around them. It's a simple yet effective tool for face detection in images, offering an intuitive user experience.
