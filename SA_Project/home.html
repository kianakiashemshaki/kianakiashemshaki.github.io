Tool Guide

This tool includes three main parts:
Page 1 (Home): Displays a map of South America with short summaries for each country.
Page 2 (Data View): Provides an interactive, zoomable chart with detailed data for every country.
Page 3 (Keyword Search): Lets you select a country and enter keywords to find specific historical, social, or economic information in a categorized table.
